## Some Information
This application is about managing tasks between the admin and users, the admin can see the completed tasks and is in progress and see all users.\
Admin can filter by word, specific user to see assigned tasks, status (complete or in-progress) and date rang.\
By logging in as a user, you will see all tasks assigned to you.\
User can show task details and add complete task after it finished and task will be removed.
